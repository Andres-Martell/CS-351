Use the arrow keys to move the location of the camera.
Use the M key to zoom in and the N key to zoom out.
Use the WASD keys to control the tilt of the camera.
Use the mouse to control the sphere's light source.
Use the X key to change between Gouraud, Phong, and no Shading
Use the C key to change between Phong and Blinn-Phong lighting
Use the 4 number key to change the material of the objects
Use the 5 number key to toggle the Mouse Light source
Use the 6 number key to toggle the Camera Light source
Use the 7 number key to toggle the Ceiling Light source
